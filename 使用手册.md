## 1. 简介

本工具叫**"误"联网原住民播客助手**，它的作用是：

> 帮你把当天的热点新闻，变成你播客节目的内容——包括文字稿和语音。

整个过程全自动，你只需要点一个按钮。它由两部分组成：

- **后端**：运行在电脑后台，负责抓新闻、生成文字稿、合成音频
- **前端**：网页界面，类似微博热搜的样子，方便你浏览和下载

---

## 2. 安装与启动

### 一键启动

双击运行项目文件夹里的 `start.bat`，它会自动：

1. 检查并安装后端依赖（Python 所需包）
2. 检查并安装前端依赖（首次运行会慢一些）
3. 同时启动后端和前端两个服务窗口

启动后会自动打开浏览器。如果没有，自动访问：

```
http://localhost:5173
```

> **注意**：启动后会弹出两个黑色命令窗口，分别是「播客助手后端」和「播客助手前端」，两个都不要关。关掉就停了。

如果以后再启动时前端依赖已安装完毕，跳过安装步骤，直接启动，速度会快很多。

## 3. 界面介绍

打开 http://localhost:5173 后，看到的页面分这几块：

### 顶部导航栏

- **今日热点**：当前页面
- **历史记录**：查看之前生成过的日期

### 日期选择器

- 点击左右箭头可以切换到之前的日期
- "今天"标签表示当前日期

### 状态指示

四条小标签显示今日内容的生成状态：

| 标签 | 含义 |
|------|------|
| 热点 | 是否已抓取热点列表 |
| 总结 | 是否已生成总结文档 |
| 文字稿 | 是否已生成播客文字稿 |
| 音频 | 是否已合成 MP3 音频 |

灰色圆点 = 未生成，绿色圆点 = 已生成。

### 热搜列表

类似微博热搜的样子，每条按热度从高到低排列。

- **排名**：前三条有特殊颜色（粉/橙）
- **热度标签**：沸 > 爆 > 热 > 新 > 商
- 点击任意一条可以查看详情

## 4. 使用方法

### 4.1 查看热点详情

打开 http://localhost:5173 后，点击任意一条热搜，会从右侧滑出一个详情面板：

- **详情摘要**标签：显示事件摘要、标签、热度条
- **文字稿**标签：显示这条事件在播客文字稿中的对应段落
- **下载**标签：下载总结文档、文字稿、音频

### 4.2 下载内容

在详情面板的「下载」标签页中，可以下载：

| 文件 | 内容 |
|------|------|
| `YYYY-MM-DD_总结文档.md` | 当天所有热点的 Markdown 总结 |
| `YYYY-MM-DD_文字稿.md` | 完整播客朗读文字稿 |
| `YYYY-MM-DD_podcast.mp3` | 语音合成音频，可直接使用 |

### 4.3 查看历史日期

点击顶部「历史记录」，会看到所有之前生成过的日期卡片。点击「查看」跳转到对应日期。

## 5. 日常使用流程

每次使用时，按以下顺序操作：

### 第一步：启动

双击 `start.bat`，等待两个黑色窗口都出现后，打开浏览器访问 http://localhost:5173。

### 第二步：生成当日内容

1. 确认日期是今天
2. 点击右上角的**「生成」**按钮
3. 等待 1~2 分钟，完成后四个状态（热点、总结、文字稿、音频）全部变绿

> 可以重复点击「生成」重新生成，会覆盖之前的内容。

### 第三步：下载使用

1. 点击任意一条热搜条目
2. 切换到「下载」标签
3. 点击「↓ 下载」按钮，获取总结文档 / 文字稿 / MP3 音频

### 第四步：停止服务

使用完毕后，关闭两个黑色命令窗口即可。所有数据已保存在 `data/` 文件夹中，下次可直接使用。

## 6. 输出内容说明

### 6.1 热点列表

每条热点包含：

- **标题**：事件名称
- **来源**：新浪新闻 / 腾讯新闻 / 微博热搜
- **摘要**：50~100 字的事件概述
- **标签**：关键词（如"财经""科技"）
- **热度值**：数值化的热度分数

### 6.2 热点总结文档（Markdown）

位置：`data/summary/YYYY-MM-DD_summary.md`

内容包括：

```
# YYYY年MM月DD日 每日热点总结

## 一、今日热点概览
[表格：排名 | 标题 | 来源 | 热度 | 标签]

## 二、热点事件详情
[每条热点的详细描述，含背景、进展、影响]

## 三、关键词标签整理
[所有标签汇总]
```

### 6.3 播客文字稿

位置：`data/script/YYYY-MM-DD_script.md`

按照主播"座机那位"的风格生成，每期结构固定：

```
[拿起座机音效]

喂。晚上好。还是我。座机那位。

今天是YYYY年MM月DD号，星期X。
咱们来聊聊今天都发生了些什么。
[停顿]

先说说今天的大概情况。
[TOP3 热点预览]

[停顿 3秒]

好，先说第一条。
—— 事件标题 ——
今天有条热搜，标题叫《XXX》。
这条消息呢，出自XXX。
……（事件口语化描述）

[停顿 2秒]

接着往下。
……（下一条）

行了。今天就聊到这儿。
您要是觉得还行，就当个心跳包，时不时回来看看。
闭眼吧。晚安。

[搁下座机音效]
```

### 6.4 语音合成音频

位置：`data/audio/YYYY-MM-DD_podcast.mp3`

使用 Windows 系统内置语音引擎合成，完全离线，音质清晰。语速偏慢，适合播客朗读。

## 7. 常见问题

### Q：启动后显示"未找到"或404？

确认后端窗口没有报错、没有关闭。刷新页面重试。

### Q：生成过程很慢？

热点抓取依赖网络，国内访问部分新闻源可能较慢，通常 1~2 分钟内完成。耐心等待即可。

### Q：热搜列表显示为空？

可能所有新闻源当日都访问失败，系统会使用内置的演示数据兜底。点击「生成」重试，或检查网络连接。

### Q：音频是英文/声音不对？

Windows 中文语音可能未安装或不是默认。TTS 会自动尝试选择中文语音，如声音不符可手动安装 Windows 中文语音包。

### Q：想改文字稿的风格？

修改文件 `backend/generator/script_writer.py`，其中的模板和语气词都在文件顶部定义，可以自行调整。

### Q：关闭后下次打开数据还在吗？

在。只要不删除 `data/` 文件夹，所有历史数据都会保留。

### Q：如何停止服务？

关闭两个黑色命令窗口（「播客助手后端」和「播客助手前端」）即可。下次使用重新双击 `start.bat`。
